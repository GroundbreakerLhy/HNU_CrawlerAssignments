* fig —— 图标文件夹

* frame_structure.png —— 方便交作业的设计图
* player.py —— 代码
* requirements.txt —— 环境依赖

## 如何使用

我的电脑装的是conda所以使用的是conda管理虚拟环境，源是清华源

首先创建虚拟环境

```powershell
conda create -n data_venv python=3.10
```

然后安装依赖

```powershell
conda activate data_venv
pip install -r requirements.txt
```

接着你运行就行了
